
## deepseen-backend

A backend for the DeepSeen project

DEV: http://localhost:2111

### Deploy

```shell script
git clone https://github.com/peterdee/deepseen-backend
cd ./deepseen-backend
nvm use 14
npm i
```

### Launch

Development:

```shell script
npm run start:dev
```

Production:
```
npm run start:prod
```
